## Description

Frontend projeto: Lista Defensoria do Estado do Rio Grande do Sul
- Front: React.js
- Framework: Next.js
- UI: radix-ui
- Icons: lucide-react
- CSS: tailwind
- Validation: useForm e zod
- Gerenciamento de estado: zustand
- Test-unitario: Jest 
- Mock: Json-server

## Defensoria do Estado do Rio Grande do Sul
 - CRUD: Modal e Form inputs use radix-ui 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# exec json-server
$ npm run mock

# dev
